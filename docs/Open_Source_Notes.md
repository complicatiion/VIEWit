# Open Source Notes

VIEWit is structured to be open-source friendly:

- No bundled CDN assets
- No telemetry
- No hidden background network service
- No required proprietary adblocking binary
- Local fallback adblocking engine
- Clear documentation of limitations
- Minimal Android permissions

Required permissions:

- `android.permission.INTERNET`
- `android.permission.ACCESS_NETWORK_STATE`

No location permission, no root access, and no VPNService are used in V1.1.2.

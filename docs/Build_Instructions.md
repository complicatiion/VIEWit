# Build Instructions

1. Install Android Studio.
2. Install Android SDK 35.
3. Open the `VIEWit` project root.
4. Let Gradle sync.
5. Build the `app` module.
6. Run on Android 10+ or Android TV.

## Build Notes

- Package name: `com.sksdesign.viewit`
- Min SDK: 29
- Target SDK: 35
- Version code: 112
- Version name: 1.1.2

## Optional AdblockPlus

Do not add an external AdblockPlus binary unless you are ready to wire a real adapter in `AdblockPlusEngine`.
